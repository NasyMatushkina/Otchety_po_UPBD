Лабораторная работа №3 по УПБД
"Использование форм для модификации данных"

Постановка задачи

Написать программу, которая позволяет просматривать, добавлять, изменять и удалять записи в таблице зарегистрированных пользователей. Для просмотра должен использоваться элемент управления ListView. Для добавления и изменения записей должны использоваться формы. Формы должны быть защищены от SQL-инъекций. 
Провести тестирование корректности работы программы.  
Написать отчёт о проделанной работе.  
P.S. Не забудьте, что если пользователь редактирует запись, то надписи на форме должны соответствовать форме редактирования, а не добавления, и наоборот. 
P.P.S. Новый пароль при создании/редактировании записи должен задаваться в явном виде, а сохраняться в базу данных - в хешированном.

Выполнение

Для выполнения этого задания используется уже ранее созданная (в ЛР2) база данных "users". Для представления данных в таблице на форме используется элемент ListView. В коллекцию добавляем две колонки "login" и "registration_data". Также на форме добавляются три кнопки "Добавить", "Изменить", "Удалить".
Для добавления и изменения данных у студентах используется новая форма InsertUpdate. 
Графическое представление программы можно увидеть в изображениях "Просмотр", "Добавление", "Изменение".
