Лабораторная работа №4 по УПБД
"Связанные элементы управления"

Постановка задания 
Спроектировать и наполнить базу данных, состоящую из трёх таблиц, последовательно связанных связью типа 1:М.
Создать программу при помощи мастера, которая на графическом интерфейсе пользователя содержит три элемента управления типа выпадающий список, предоставляющих пользователю возможность выбора из элементов соответствующей таблицы, относящихся лишь к выбранному на более высоком уровне иерархии элементу.  
Провести тестирование корректности работы программы.  
Написать отчёт о проделанной работе.

Решение
Для решения была использована ранее созданная база данных (в ЛР1) "Universities".
В ходе выполнения были созданы такие объекты, как bindingSource (для каждой таблицы), SQLiteDataAdapter (также для каждой таблицы), dataSet. При загрузке формы создается три объекта типа SQLiteDataAdapter с командой SELECT *. Данные, полученные после выполнения команд, записываются в dataSet1. Также устанавливаются отношения между таблицами с помощью свойства dataSet1.Relations.Add(). Далее происходит установка свойств DataSource и DataMember для каждого объекта типа BindingSource. И, наконец, данные выводятся в каждый comboxBox.
